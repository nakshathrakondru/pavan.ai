# Pavan – Air Quality Monitoring System

## Overview

Pavan is a web application designed to help users understand the air quality of a selected location through a clean and intuitive interface.
The system focuses on presenting environmental data in a simple and readable format so that users can quickly interpret pollution levels and make informed decisions.

---

## Features

* Location-based air quality display
* Clean and responsive user interface
* Structured backend using Flask
* Data presented in an easy-to-understand format

---

* Flask

## How to Run Locally

1. Clone the repository

```
git clone https://github.com/nakshathrakondru/Pavan.AI.git
```

2. Navigate into the folder

```
cd Pavan.AI
```

3. Install dependencies

pip install -r requirements.txt

4. Run the application

python app.py
